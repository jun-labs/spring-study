## Getting Started

> You should install jdk 17 or higher. <br/>
> You must set up the application.yml.

<br/><br/><br/>

## Run Application

````text
./gradlew bootRun
````

<br/><br/>

## Run Test

````text
./gradlew test
````

<br/><br/>

## Run Build

````text
./gradlew build
````

<br/><br/>

## Result

You can see the Swagger UI by running the application and accessing `http://localhost:8080/swagger-ui/index.html`.
